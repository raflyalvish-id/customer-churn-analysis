# Customer Churn Analysis and Visualization

## 📊 Overview
This project analyzes customer churn behavior using Python to understand patterns, identify key factors influencing churn, and provide actionable business insights.

## 🎯 Objectives
- Compare churn vs non-churn customer distribution  
- Analyze the relationship between customer tenure and total charges  
- Identify top cities with the highest churn rates  
- Understand the distribution of tenure across churn categories  
- Provide data-driven insights and recommendations  

## 🛠 Tools & Technologies
- Python (Pandas, NumPy)  
- Data Visualization (Matplotlib, Seaborn)  

## 📂 Dataset
- Telco Customer Churn Dataset (included in this repository)

## 🔍 Key Analysis
- Churn vs Non-Churn percentage comparison  
- Tenure vs Total Charges relationship  
- Top 5 cities with highest churn and non-churn customers  
- Distribution analysis of tenure by churn status  

## 📈 Key Insights
- Customers with shorter tenure are more likely to churn  
- Higher total charges are associated with longer customer retention  
- Certain cities show significantly higher churn rates  
- Customer retention increases as tenure increases  

## 💡 Recommendations
- Focus retention strategies on new customers (low tenure)  
- Provide targeted offers for high-risk churn segments  
- Analyze regional strategies in high-churn cities  
- Improve onboarding experience to increase early retention

2. Run all cells
